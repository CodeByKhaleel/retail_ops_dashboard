# Contributing to Retail Ops Dashboard

Thank you for contributing to the Retail Ops Dashboard project.

## Development Setup

1. Clone the repository

```bash
git clone https://github.com/CodeByKhaleel/retail_ops_dashboard.git
```

2. Install dependencies

```bash
npm install
```

3. Start development server

```bash
npm run dev
```

---

## Branch Naming Convention

Use descriptive branch names:

* `feature/add-realtime-charts`
* `fix/socket-reconnection`
* `improvement/dashboard-performance`

---

## Commit Message Convention

Examples:

```bash
feat: add websocket analytics updates
fix: resolve inventory sync issue
docs: update setup instructions
```

---

## Pull Request Process

Before submitting a PR:

* Ensure code builds successfully
* Test new features locally
* Update documentation if needed
* Keep pull requests focused and small

---

## Coding Guidelines

* Use TypeScript wherever possible
* Follow consistent formatting
* Write reusable components/services
* Avoid hardcoded configuration values
* Keep backend services modular

---

## Reporting Issues

Please use the provided issue templates for:

* Bug reports
* Feature requests

---

Thank you for helping improve the project.
